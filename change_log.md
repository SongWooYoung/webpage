# 변경 이력 (Change Log)

## 2025-12-27

### Study 섹션 학습 내용을 개별 페이지로 분리

**변경 요청**: Study 섹션의 학습 내용이 한 페이지에 모두 표시되는 것이 아니라, 목록으로 보이고 클릭 시 개별 페이지로 이동하도록 변경

**변경 내용**:
- Study 도메인별 인덱스 페이지는 학습 항목 목록만 표시
- 각 학습 내용은 개별 HTML 파일로 분리
- 날짜 기반 파일명 사용 (예: 2025-12-27-web-build.html)

**변경 이유**: 학습 내용이 누적되면서 한 페이지가 너무 길어지는 것을 방지하고, 개별 학습 내용 접근성 향상

---

### Home 섹션에 프로필 사진 공간 추가

**변경 요청**: 개인 소개 섹션의 오른쪽 공간에 프로필 사진을 추가할 수 있는 영역 마련

**변경 내용**:
- 개인 소개 섹션을 2열 레이아웃으로 변경 (왼쪽: 정보, 오른쪽: 사진)
- 프로필 사진 영역 추가
- 반응형 디자인 적용 (모바일에서는 세로 배치)

**변경 이유**: 개인 소개 섹션의 빈 공간을 활용하여 프로필 사진을 표시하기 위함

---

### Home 섹션에 개인 소개 추가

**변경 요청**: Home 페이지에 개인 정보(이름, 이메일, 출신학교 등)를 표시하는 섹션 추가

**변경 내용**:
- Home 섹션에 "개인 소개" 영역 추가
- 이름, 이메일, 소속/학교, 전공 필드 포함
- 선택적으로 관심 분야 및 소셜 링크 추가 가능한 구조

**변경 이유**: 웹사이트 방문자가 사이트 소유자의 기본 정보를 확인할 수 있도록 하기 위함
