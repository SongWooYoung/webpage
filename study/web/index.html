📌 1. 작업 배경

이 프로젝트는 개인 학습·계획·일기·취미·성과를 장기적으로 기록하기 위한 정적 웹사이트를 만드는 것이 목적이다.

구현은 HTML / CSS / JavaScript 기반으로 수행하며, 요구사항 정의·변경·구현을 Agent와 협업하는 spec-driven 개발 방식으로 진행한다.

📁 2. 기본 설계 및 문서 체계 확정

다음 3개의 핵심 문서를 중심으로 프로젝트를 운영하기로 확정하였다.

2.1 what2do.md

  • 웹사이트의 최종 요구사항 명세
  • Single Source of Truth
  • "현재 기준에서 무엇을 만들어야 하는지"만 기록
  • 변경 이력이나 이유는 기록하지 않음

2.2 agent_rules.md

  • Agent의 역할과 행동 규칙 정의
  • 문서 우선순위 및 수정 절차 명시
  • Agent를 "자율적 설계자"가 아닌 명세 기반 구현자로 제한

2.3 change_log.md

  • 요구사항 변경이 발생했을 때만 기록
  • append-only
  • 초기 상태에서는 비어 있는 것이 정상
  • 변경의 기준선(baseline) 역할

⚙️ 5. Cloudflare Pages 배포 과정 및 문제 발생

5.1 Cloudflare Pages 설정

수행한 설정:

  • Cloudflare Dashboard 접속
  • Workers & Pages → Pages 프로젝트 생성
  • Create application → Pages → Connect to Git
  • GitHub 저장소 연결: SongWooYoung/webpage

배포 설정:

  • Framework preset: None
  • Build command: (비워둠)
  • Build output directory: /
  • Root directory: /

5.2 배포 중 오류 발생

첫 배포 시 발생한 문제:

  • ❌ Cloudflare Pages가 정적 자산을 찾지 못함
  • ❌ 루트 경로 인식 문제 발생
  • ❌ Pages가 프로젝트 구조를 명확히 인식하지 못함

✅ 6. wrangler.toml 도입을 통한 배포 오류 해결

문제 원인

Cloudflare Pages가 프로젝트의 정적 자산 위치를 명확히 알 수 있는 설정 파일이 없음

해결 방법

wrangler.toml 설정 파일 추가

wrangler.toml의 역할

  • 프로젝트 이름 및 정적 자산 위치 명시
  • Cloudflare Pages가 어떤 디렉터리를 서빙해야 하는지 명확히 지정

결과

  • ✅ Pages가 정적 자산을 정상적으로 인식
  • ✅ 배포 오류 해결
  • ✅ 이후 Git push 시 자동 재배포 안정화

👤 7. Home 섹션 요구사항 변경 – 자기소개 추가

7.1 변경 배경

일반적인 개인 기록용 웹사이트/블로그에는 Home 페이지에 기본적인 자기소개(Profile) 정보가 필요하다고 판단하였다. 이는 초기 명세에 없던 내용이므로 요구사항 변경으로 분류하였다.

7.2 요구사항 변경 절차 수행

  1. change_log.md 기록
  2. what2do.md 수정 (Home 섹션 명세 업데이트)
  3. 코드 구현 (HTML / CSS 수정)

📝 8. Home 섹션에 추가된 자기소개 구성

포함된 정보 필드

  • 이름 (Name)
  • 이메일 (Email)
  • 소속 / 학교 (Affiliation / School)
  • 전공 (Major)
  • 관심 분야 (Interests / Research Interests)
  • 외부 링크 (GitHub, LinkedIn)
  • 프로필 사진 (이미지)

이는 개인 기록 웹사이트의 "정체성"을 드러내는 영역으로 사용된다.

📂 9. 수정된 파일 목록

  • index.html → Home 페이지에 개인 소개 섹션 추가
  • style.css → 프로필 영역 레이아웃 및 스타일 추가
  • what2do.md → Home 섹션 요구사항 명세 업데이트
  • change_log.md → 요구사항 변경 이력 기록

🔄 10. Git 커밋 및 재배포

git add what2do.md change_log.md index.html css/style.css
git commit -m "Add personal profile section to Home page"
git push origin main
